🎯 Higher Lower Game (Flask)

How it works:

The user guesses a number by entering it in the URL Like http://127.0.0.1:5000/1 or http://127.0.0.1:5000/100
The app compares the guess with a random number
The app responds with:
Too Low
Too High
Correct
