🎯 Guess the Number – Flask Web App

A simple web-based number guessing game built using Flask.

🚀 How it works
The server generates a random number between 0 and 100
The user guesses the number by typing it in the URL
The app responds with:
"Too High" 📈
"Too Low" 📉
"Correct!" 🎉
🧠 What I learned
Building a web server with Flask
Creating dynamic routes using URL parameters
Handling user input from the browser
Using conditional logic to control application flow
Generating random numbers in Python
Returning HTML and images from a Flask app
🛠️ Technologies used
Python
Flask
▶️ How to run
pip install flask
python app.py

Then open your browser and go to:

http://127.0.0.1:5000/
📸 Demo

Try entering a number in the URL like:

http://127.0.0.1:5000/50
